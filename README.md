# 🧠 AI Mental Health Support System

An AI-powered mental health support application built with Streamlit that analyzes user conversations and detects emotional patterns using Machine Learning and Natural Language Processing (NLP). The system provides intelligent responses and emotion-based interaction to support mental wellness conversations.

## 📂 Files Included

* `app.py` — Main Streamlit application
* `emotion_model2.pkl` — Trained emotion detection model
* `tfidf_vectorizer2.pkl` — TF-IDF vectorizer for text preprocessing
* `mental_health_conversations.csv` — Dataset used for training
* `CODE_OF_TRAINING_MODEL_FOR_MOTION_DETECTION.pdf` — Training model documentation

## 🚀 Features

* Emotion detection from text conversations
* NLP-based text preprocessing
* Real-time interaction using Streamlit
* Machine Learning prediction system
* User-friendly and responsive interface

## 🧠 Technologies Used

* Python
* Streamlit
* Scikit-learn
* NLP (TF-IDF)
* Pandas & NumPy
